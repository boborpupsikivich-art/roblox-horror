# Draugr's Mound

Psychological multiplayer horror game for Roblox. Two players explore a cursed Viking burial mound, solving runic puzzles while a draugr watches from the shadows.

Built with Luau + Rojo. Scandinavian folklore setting.

## Setup

1. Install [Roblox Studio](https://create.roblox.com)
2. Install Rojo: `brew install rojo`
3. Clone → `rojo serve` → connect in Studio
4. Build the map (see `assets/map-building-guide.md`)
5. Play!

See `CLAUDE.md` for full architecture and context.
